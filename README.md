# API Testing Framework

A Python-based API testing framework using pytest for testing WooCommerce APIs.

## Project Structure

- **src/** - Source code containing core functionality
  - **configs/** - Configuration management
  - **dao/** - Data Access Objects for database operations
  - **data/** - Test data and payloads
  - **helpers/** - Helper functions for various API operations
  - **utilities/** - Utility functions for credentials, database, requests, and API interactions

- **tests/** - Test suites organized by feature
  - **coupons/** - Coupon-related tests
  - **customers/** - Customer-related tests
  - **orders/** - Order-related tests
  - **products/** - Product-related tests
  - **test_healthcheck.py** - Initial health check test
